### GitHub Issues

If the documentation doesn't answer your question, open a
[GitHub Issue](https://github.com/dterracino/color_tools/issues). Please:

- Search existing issues before opening a new one
- Use the appropriate issue template (Bug Report or Feature Request)
- Include your Python version and `color-match-tools` version

### Security Issues

Do **not** report security vulnerabilities as public issues. See
[SECURITY.md](SECURITY.md) for how to report them privately.

## Scope of Support

This project is maintained by a single developer in spare time. Support is provided
on a best-effort basis. Response times may vary.

What is in scope:

- Bugs in color conversions, distance metrics, or CLI behavior
- Incorrect data in the color or filament databases
- Documentation gaps or errors
- Feature requests for color science functionality

What is out of scope:

- General Python programming questions
- Help with your specific application's architecture
- Issues caused by unsupported Python versions (< 3.10)
